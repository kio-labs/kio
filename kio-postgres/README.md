# kio-postgre (In Developing)
postgresql protocol implementation using [kio-async](https://github.com/kio-labs/kio-async).

> [!WARNING]
> This project is experimental and not intended for production use.
> It is mainly for learning, testing, and personal experiments.

## How to run
- native (Only supoort macos now): `./gradlew :example:postegre-client:runDebugExecutableMacosArm64`
